Purpose of the demo

Using JavaScript to filter two lists

In this demo we will play with a backend-service written with Java Springboot and MongoDB, Twitter Bootstrap, RxJS Observable and some plain JavaScript (TypeScript) generated by Angular 6 Click the Profiles link in the menu to start Note! The demo is best viewed on a wide screen in order to compare the lists in three columns
Future versions will show sorting lists, displaying the data in an OpenLayers map and more
